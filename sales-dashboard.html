✅ Checklist Sebelum Menghubungkan
☐  Google Sheets sudah dibuka publik: Klik di Sheets → kanan atas "Share" → Anyone with link → Viewer
☐  Sheet sudah di-publish: File → Share → Publish to web → pilih sheet → CSV → Publish
☐  URL mengandung pub?output=csv atau gviz/tq?tqx=out:csv
☐  Ketiga URL sudah diisi (MASTER_TOKO, SALES_DAILY, SALES_TARGET)
Kolom MASTER_TOKO: kode_toko, nama_toko, area, kota, cluster, operation_manager, arco, store_leader

Kolom SALES_DAILY: tanggal (YYYY-MM-DD), kode_toko, uang_masuk_total, transaksi_total, setoran_tunai, debit_bca, kk_bca, qr_bca, debit_mandiri, kk_mandiri, qris_mandiri, debit_bri, kk_bri, qris_bri, qr_shopee, mds, gm, selisih_kasir, selisih_mds_gm

Kolom SALES_TARGET: kode_toko, target_sales_harian (Sales Rp per hari)

Cara publish ke CSV: Google Sheets → File → Share → Publish to web → Pilih sheet → Format CSV → Publish → Copy URL
